# 可愛方塊屋 (Komorebi Craft) - 遊戲規劃書

## 1. 遊戲概述
《可愛方塊屋》是一款以 Minecraft (Voxel) 風格為靈感，結合類似《動物森友會》佈置樂趣的 3D 立體空間設計遊戲。玩家可以在寬廣的草地上，自由擺放、旋轉、縮放各種主題物件，打造專屬於自己的療癒小天地。

## 2. 核心技術架構
- **前端框架**: React 19 + TypeScript
- **3D 渲染**: React Three Fiber (Three.js) & @react-three/drei
- **介面樣式**: Tailwind CSS (Vibrant Palette 主題)
- **動畫效果**: Framer Motion (motion/react)
- **圖示庫**: Lucide React

## 3. 遊戲功能特色

### A. 3D 擺設機制
- **多維度控制**: 選取物件後，可通過 UI 面板進行：
  - **移動**: 前後左右移動，以及**垂直高度調整** (實現懸空效果)。
  - **旋轉**: 以 45 度為單位精確旋轉。
  - **縮放**: 自由調整物件大小，適應不同的造景需求。
  - **複製**: 一鍵複製選取中物件，完整保留其縮放比例與高度設定。
  - **刪除**: 移除不需要的物件。

### B. 四大主題物件庫
1. **和風・櫻花祭 (Sakura Theme)**
   - 櫻花樹、紅橋、燈籠、鳥居、草地、碎石步道。
2. **療癒・動物園 (Zoo Theme)**
   - 熊貓、大像、小狗、小貓、竹子、圍欄。
   - **人物角色**: 老爺爺、老奶奶、男人、女人、小男孩、小女孩。
3. **壯麗・富士山 (Fuji Theme)**
   - 富士山、神社、松樹、景石、空中雲朵、湖泊。
4. **摩登・城市 (City Theme)**
   - 台北101大樓、商業大樓、公寓、一般住家、便利商店。
   - 大型公園、馬路、紅綠燈、路燈、汽車、機車。

### C. 智慧設計、環境模擬與存檔管理
- **AI 智慧構圖 (AI Composition)**: 使用者輸入自然語言描述（如「繁忙的台北街頭」或「安靜的熊貓竹林」），AI 設計師將自動選擇物件並進行美觀布局（限 100 件內）。
- **天氣動態模擬 (Weather Simulation)**:
  - **晴天 (Sunny)**: 高對比度、明媚陽光。
  - **雨天 (Rainy)**: 灰暗雲層、流動的雨滴粒子效果。
  - **下雪 (Snowy)**: 白色地面、緩慢飄落的雪花效果。
- **背景音樂 (Background Music)**: 播放輕快療癒的 Lo-fi 音樂，提升佈置時的沉浸感。
- **即時拍照 (Screenshot)**: 擷取當前 3D 畫面並下載為 PNG。
- **配置匯出 (Export)**: 將當前所有擺設位置與屬性匯出為 JSON 檔案。
- **配置匯入 (Import)**: 讀取先前的 JSON 存檔，瞬間還原場景。
- **一鍵清除 (Clear All)**: 清空所有物件，從零開始創作。

## 4. 操作指南 (User Controls)
- **放置物件**: 在目錄選取物件後，於地面點擊 **[滑鼠右鍵]**。
- **選取物件**: 在 3D 畫面中對物件點擊 **[滑鼠左鍵]**。
- **取消選取**: 點擊地面的空白處 **[滑鼠左鍵]**。
- **視角旋轉**: 按住 **[滑鼠左鍵]** 並拖拽。
- **視角平移**: 按住 **[滑鼠右鍵]** 並拖拽。
- **畫面縮放**: 使用 **[滑鼠滾輪]**。

## 5. 設計風格
- **色彩**: 採用 "Vibrant Palette" (薄荷綠、櫻粉色、奶油黃)。
- **字體**: 使用粗體黑體 (Blackweight) 的日系可愛與現代感結合。
- **質感**: Voxel 像素風格建模，搭配環境光 (Ambient Light) 與柔和陰影 (Contact Shadows)，營造自然清新的視覺感受。
